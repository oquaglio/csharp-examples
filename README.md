# csharp-examples
C# examples

## Setup

To get these working in VSCode, just install the extension "C# Dev Kit" and a dotnet sdk.

## Build

You can build everything together:

```SH
dotnet build CSharpExamples.sln
```

## Add More Projects

In VSCode:

f1 -> .NET: New Project -> Console App ...

Or:

dotnet sln add
